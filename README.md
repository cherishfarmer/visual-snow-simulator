# Visual Snow Simulator

An interactive web tool to help people understand what Visual Snow Syndrome (VSS) looks like.

## What is Visual Snow Syndrome?
VSS is a neurological condition affecting 2-3% of the population, characterized by persistent visual disturbances similar to TV static.

## Features
- Upload any image to see it with visual snow
- Adjust density, speed, size, and color of snow particles
- Compare original vs simulated side-by-side
